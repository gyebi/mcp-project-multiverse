# Blue Report MCP Server

This project is a Python MCP server built with FastMCP. It provides tools for fetching Blue Report stories from hourly, daily, and weekly endpoints.

## Requirements

- Python 3.10 or higher
- uv
- FastMCP
- requests
- beautifulsoup4

## Setup

Create and activate the virtual environment:

```bash
uv init
uv add fastmcp requests beautifulsoup4# mcp-project-multiverse
